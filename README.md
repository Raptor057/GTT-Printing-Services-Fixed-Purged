# Documentación del Proyecto

  

## Introducción

  

La función de este proyecto es imprimir etiquetas individuales a las impresoras de WB (Zebra)
  

## Estado del Proyecto

  

En constantes modificaciones
  

## Objetivos
- Imprimir etiquetas individuales de las transmisiones en las impresoras zebra de producción. 

  

## Tecnologías Utilizadas

- [.NET Framework 4.7.2](https://dotnet.microsoft.com/es-es/download/dotnet-framework/net472)
- [DevExpress](%5C%5Cmxsrvdata%5CDATA%5CIT%5CSoftware%5CDevExpressComponentsBundle-17.1.17.exe) `\\mxsrvdata\DATA\IT\Software\DevExpressComponentsBundle-17.1.17.exe`


  

## Arquitectura del Proyecto

- Sin arquitectura

  

## Funcionalidades

  - Imprimir etiquetas individuales para las transmisiones, la información la envía la base de datos Apps 

  

## Instalación

Este proyecto se abre con Visual Studio 2019 con el DevExpress instalado previamente y se publica en la ubicación (`\\mxsrvapps\apps\web\gtservices\printing`)

  

## Uso

  

  - Imprimir etiquetas individuales para las transmisiones
